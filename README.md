## 快速使用 / Quick Usage

- ### 使用脚本 / Use Script

  ```shell
  git clone https://github.com/PIKACHUIM/DockerFiles.git
  cd DockerFiles && chmod +x -R *.sh
  ./Manager.sh
  ```

  1. Enter  `c` and press `Enter`
  2. Choose system and desktop
  3. Use SSH/NoMachine to connect docker
